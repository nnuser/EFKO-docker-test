
1. Установить docker и docker-compose
2. Перейти в папку yii2-app-basic и склонировать базовый проект yii2 "git clone https://github.com/yiisoft/yii2-app-basic.git"
3. Вернуться в корень и выполнить "sudo docker-compose up --build"

Если необходимо скопировать базу в "databases" или просто создать базу yii2basic с помощью adminer "http://127.0.0.1:6080"
Проект будет доступен по адресу "http://127.0.0.1:8080"
